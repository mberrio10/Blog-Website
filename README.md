# Blog Page Template Layout

This template was develop using create-react-app boilerplate created by for 4Geeks Academy. this is a template i created for future projects involving a blog page, full-responsive, and developed using HTML5,CSS3, Bootstrap, and React.js. you can dowload this repo overwrite and customize anything to your needs.

#### Download the boilerplate using git

```
$ git clone https://github.com/mberrio10/Blog-Website.git
$ cd react-hello
```

##### and install the npm package:
```
$ npm install
```
## Customize and start to see your changes!

For Windows, Mac, Linux or Gitpod, start the webpack server with live reload:
- `$ npm run start`

You can update the `styles/index.scss` or `js/index.js` depending on your needs.
Add more files into your, `./src/js/components` or styles folder as you need them.
